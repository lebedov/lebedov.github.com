CULA Routines¶

Helper Routines¶

culaCheckStatus Raise an exception corresponding to the specified CULA status
culaGetErrorInfo Returns extended information about the last CULA error.
culaGetLastStatus Returns the last status code returned from a CULA function.
culaGetStatusString Get string associated with the specified CULA error status code.
culaInitialize Must be called before using any other CULA function.
culaSelectDevice Selects a device with which CULA will operate. Must be called
culaShutdown Shuts down CULA.
