scikits.cuda.special.sici¶

scikits.cuda.special.sici(x_gpu, dev)¶

Sine/Cosine integral.

Computes the sine and cosine integral of every element in the input matrix.

Parameters :

x_gpu : GPUArray

Input matrix of shape (m, n).

dev : pycuda.driver.Device

Device object to be used.
Returns :

(si_gpu, ci_gpu) : tuple of GPUArrays

Tuple of GPUarrays containing the sine integrals and cosine integrals of the entries of x_gpu.

Examples

>>> import pycuda.gpuarray as gpuarray
>>> import pycuda.autoinit
>>> import numpy as np
>>> import scipy.special
>>> import special
>>> x = np.array([[1, 2], [3, 4]], np.float32)
>>> x_gpu = gpuarray.to_gpu(x)
>>> (si_gpu, ci_gpu) = sici(x_gpu, pycuda.autoinit.device)
>>> (si, ci) = scipy.special.sici(x)
>>> np.allclose(si, si_gpu.get())
True
>>> np.allclose(ci, ci_gpu.get())
True
