scikits.cuda.misc.shutdown¶

scikits.cuda.misc.shutdown()¶

Shutdown libraries used by scikits.cuda.

Shutdown the CUBLAS and CULA libraries used by high-level functions provided by scikits.cuda.

Notes

This function does not shutdown PyCUDA.
