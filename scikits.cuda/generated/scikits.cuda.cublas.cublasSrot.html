scikits.cuda.cublas.cublasSrot¶

scikits.cuda.cublas.cublasSrot(n, x, incx, y, incy, sc, ss)¶

Apply a real rotation to a real matrix.

Multiplies the single-precision matrix [[sc, ss], [-ss, sc]] with the 2 x n single-precision matrix [[x.T], [y.T]].

Parameters :

n : int

Number of elements in input vectors.

x : ctypes.c_void_p

Pointer to single-precision input/output vector.

incx : int

Storage spacing between elements of x.

y : ctypes.c_void_p

Pointer to single-precision input/output vector.

incy : int

Storage spacing between elements of y.

sc : numpy.float32

Element of rotation matrix.

ss : numpy.float32

Element of rotation matrix.

Notes

Both x and y must contain n elements.
