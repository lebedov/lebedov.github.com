scikits.cuda.special.exp1¶

scikits.cuda.special.exp1(z_gpu)¶

Exponential integral with n = 1 of complex arguments.

Parameters :

z_gpu : GPUArray

Input matrix of shape (m, n).
Returns :

e_gpu : GPUArray

GPUarrays containing the exponential integrals of the entries of z_gpu.

Examples

>>> import pycuda.gpuarray as gpuarray
>>> import pycuda.autoinit
>>> import numpy as np
>>> import scipy.special
>>> import special
>>> z = np.asarray(np.random.rand(4, 4)+1j*np.random.rand(4, 4), np.complex64)
>>> z_gpu = gpuarray.to_gpu(z)
>>> e_gpu = exp1(z_gpu, pycuda.autoinit.device)
>>> e_sp = scipy.special.exp1(z)
>>> np.allclose(e_sp, e_gpu.get())
True
